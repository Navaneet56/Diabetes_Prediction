# Diabetes Prediction Web App

An end-to-end machine learning web application that predicts the likelihood of diabetes based on clinical metrics. 

## 🌐 Live Demo
Check out the live interactive app here: https://diabetesprediction-aespcamdwnidxydegvvcas.streamlit.app/

## 🛠 Tech Stack
* Python
* Scikit-Learn
* Streamlit
* GitHub Cloud Deployment

## 📂 Project Structure
* `Diabetes_Prediction.ipynb` - Data analysis & model training
* `app_diabetes.py` - Streamlit application interface
* `diabetes_model.pkl` - Trained prediction model
* `diabetes_scaler.pkl` - Data normalization scaler
* `requirements.txt` - Python package dependencies
